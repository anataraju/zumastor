Zumastor Walkthroughs

Ubuntu walkthrough on VMWare Player

First, install vmware player on your machine. (If you're using Ubuntu, it might be as simple as apt-get install vmware-player.)

Second, download a vmware image with Ubuntu preinstalled. I'm not sure what the best is, but the oddly-named site www.thoughtpolice.co.uk/vmware seems reasonable. I chose the Ubuntu Server 7.10 download, clicked on the "fast torrent!" link, and told Firefox to open the torrent file; this popped up kTorrent, which downloaded the 189 megabyte image quickly. The image is several files zipped together, so I unpacked it with 

$ mkdir ~/vmubuntu
$ cd ~/vmubuntu
$ unzip ~/ubuntu-server-7.10-i386.zip

Third, start up vmware player (Applications / System Tools / VMWare Player) and tell it to load the .vmx file in the directory where you unpacked the vmware image.

Fourth, inside vmware, log in to the virtual ubuntu box (using the username and password from the site where you downloaded the image).

Fifth, inside vmware, download and install the Zumastor packages, e.g. 

$ sudo apt-get install links
$ linkx zumastor.org
[download the kernel image, zumastor, and ddsnap packages]
$ sudo dpkg -i ddsnap zumastor kernel
This currently fails.

Sixth, send annoyed email to the zumastor developers, asking them to try this walkthrough :-)
